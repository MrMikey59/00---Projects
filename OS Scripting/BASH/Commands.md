# BASH Commands

| Command | Description |  
| --- | --- |  
| cat \<FileName> | List contents of the file. |  
| cd \<Path> | change directory |  
| ls | List structure (folder) contents <BR> ` ls -a` - shows in wide mode |  
| md \<Path> | Make a new directory |  
| rd \<Path> | remove directory |
